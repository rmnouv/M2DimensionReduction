## Overview

This project implements and compares 4 Independent Component Analysis (ICA) algorithms for blind source separation:

| Algorithm | Type | Score Function |
|:---|:---|:---|
| **Infomax Batch** | Full-batch gradient ascent | Adaptive kurtosis switching |
| **FastICA** | Fixed-point (sklearn) | log cosh contrast |
| **SGD-ICA** | Stochastic gradient descent | Adaptive kurtosis switching |
| **EM-ICA** *(creative contribution)* | Stochastic EM | **Learned via Mixture of Gaussians** |

All gradient-based algorithms share the same update rule on whitened data (course Section 5.6.3):

```
V ← V + η (V + (1/N) g(Y) X^T)
```

followed by symmetric decorrelation to maintain orthogonality.

## Key Results

- All algorithms achieve Amari < 0.02 on synthetic, audio, and image data
- **EM-ICA outperforms on real photographs** (Amari 0.034 vs 0.086) — sources have similar kurtosis, so kurtosis switching fails but MoG learns the density shapes
- All algorithms **fail on Gaussian sources** (Amari > 0.3), confirming the ICA identifiability theorem
- FastICA is ~500x faster than gradient methods due to quasi-Newton convergence
- On real EEG, all algorithms identify one component with kurtosis ≈ 45 (likely eye blink artifact)
